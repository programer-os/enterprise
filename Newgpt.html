Explicaciones rápidas

  • margin: espacio exterior. Ej: margin:16px;
  • padding: espacio interior. Ej: padding:12px;
  • border: borde (grosor estilo color). Ej: border:2px solid #ddd;
  • box-shadow: sombra (x y blur spread color).
  • linear-gradient(angle, color1 stop%, color2 stop%): gradiente lineal. El ángulo usa deg.
  • m-: si ves m- en clases (ej. m-4) es de Tailwind CSS (shorthand para margin). No es CSS puro; en CSS normal se usa margin.
